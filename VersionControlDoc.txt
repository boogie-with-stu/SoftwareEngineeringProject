Version Control Documentation
v 1.0
------------------------------

How to setup the project on your machine:
0.) Make sure Git is properly setup and installed on your computer.
1.) Login into https://www.github.com.
  a.)  If you do not have account, please register for an account.
2.) Once logged in, navigate to the main page of the repository.
3.) Under the repository name, click Clone or download.
4.) In the Clone with HTTPs section, click copy the cline URL for the repository.
5.) Open the command line where Git is configured, for windows that will probably be Git Bash.
6.) Type git clone, and then paste the URL you copied in Step 2.
	git clone https://github.com/ProjectName
7.) Press enter and the clone will be created.

How to edit the project or add files:
1.) Once you have the project cloned to your computer, run a git fetch in your git command line.
2.) Git fetch will make sure that you have all the latest changes.
3.) Run a git add . to add all of your changes.
4.) Next run a git commit -m "Your message here" to commit your changes.
5.) Type in git push origin master to push your changes to the repository.
